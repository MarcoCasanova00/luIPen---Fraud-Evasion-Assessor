# Frontend Demo Source

This directory contains the source files for the frontend demo.

## Structure

```
frontend-demo/
├── src/
│   ├── index.html      # Main demo page
│   ├── style.css       # Styles
│   └── script.js       # Client-side logic
└── build/              # Built output (copy to docs/)
```

## Building

Run `scripts/build_demo.sh` to copy files to `docs/` directory.

## Deployment

The `docs/` directory is configured for GitHub Pages deployment.
